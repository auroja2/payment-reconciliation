# Distilled Prompt

Build a reconciliation tool for a payments company to compare its platform transaction records with bank settlement records at month end. Generate synthetic test data with these gap types: (1) settlement in next month, (2) rounding difference (shows only when summed), (3) duplicate entry in one dataset, (4) refund with no matching original transaction. State all assumptions. Implement backend logic to find and report all gaps, and a professional, modern frontend UI to display results. Include test cases for all gap types.
